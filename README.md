# 🏆 Maratona de Programação

Repositório com as soluções dos exercícios que estou resolvendo durante os treinos de maratona de programação na faculdade.

## Sobre

Sou estudante de Ciência da Computação e estou usando esse repositório para acompanhar minha evolução ao longo dos treinos semanais. Os problemas são resolvidos principalmente em **C++**, com foco em algoritmos e estruturas de dados.

A maioria dos exercícios vem de juízes online como o [Beecrowd](https://www.beecrowd.com.br).

## Organização

```
maratona-programacao/
├── 2025/
│   ├── semana-1/
│   ├── semana-2/
│   └── ...
└── README.md
```

Cada pasta de semana contém os arquivos `.cpp` com as soluções dos problemas resolvidos naquele período.

## Tecnologias

- **Linguagem:** C++20
- **Compilador:** G++ / MinGW
- **Juiz online:** Beecrowd

## Tópicos abordados

- Estruturas de dados (vector, set, map, stack, queue)
- Ordenação e busca
- Strings e manipulação de input
- Algoritmos gulosos
- E muito mais conforme o treino avança...
